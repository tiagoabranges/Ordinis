# ORDINIS — Contexto do Produto

ORDINIS é um sistema de controle financeiro pessoal com foco em organização, velocidade e clareza.

A aplicação é um monorepo com:

- apps/api: backend em NestJS
- apps/web: frontend em React + Vite
- PostgreSQL via Docker
- Prisma ORM
- autenticação JWT
- Swagger
- módulos financeiros já existentes no backend

O objetivo atual é implementar rapidamente a primeira versão funcional da planilha financeira.

A experiência principal deve lembrar uma planilha moderna:

- rápida
- organizada
- com edição simples
- com filtros
- com resumo financeiro
- com possibilidade de expandir parcelamentos

O foco agora é MVP funcional, não IA, não automações avançadas e não features complexas.
